Follow these steps to be able to run the program properly:

1) Download and Install any Java IDE (Recommended: NetBeans 8.2, Eclipsce), along with JDK (Java Development Kit)
and JRE (Java Running Environment). (JDK and JRE can be packed with the IDE when downloading it).
2) Install the IDE.
3) Open the project using "File", then "Open Project".
4) Choose the project.
5) Press the "Debug" button or Ctrl+F5 on the keyboard to compile and build first.
6) Press the "Run" butoon or F5 on the keyboard to run the project.

If the program is in ".jar" extension, then you don't have to download an IDE. All what you need to do is download
and install the latest JRE to run the program.
